When you've cloned the repo 
1. Open xampp and start apache go to phpMyAdmin and create a database with the name *wsdprojectyii2*
2. Open up the terminal and run these command. *composer install* followed by *php yii migrate*
3. go to http://localhost/wsdproject/backend/web/index.php?r=site%2Flogin the credentials are 
usr: admin@website.com
pw: MoreandMore#789
4. Create a product, from there you can view, edit or delete
5. Go to http://localhost/wsdproject/frontend/web/index.php as a guest user, you can either sign up, contact or view the products the admin created.
